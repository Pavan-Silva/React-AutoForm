# Examples

This folder contains small example apps demonstrating how to use React AutoForm.

- `basic` — a minimal Vite + React example showing default renderers and how to override them.
- `wizard` — example demonstrating the `AutoFormWizard` component with a wizard-style flow.

Run one of the examples:

```bash
cd examples/basic         # or "wizard" to see the wizard demo
npm install
npm run dev
```
